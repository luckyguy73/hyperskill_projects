Extracting words from a text Open on JetBrains Academy

You need to write a program to extract all words that include the "program" substring from a given text. The text can be large enough. You should not take whitespaces, punctuation marks and other special characters as parts of the extracted word.

Try to use Matcher and Pattern to solve it. All imports needed are already here. Your pattern should be case insensitive.

Input data format

A single string representing a text about programs, programmers or programming.

Output data format

Multiple lines where each line contains two parts separated by one space: the start index of a word and the word itself.


Sample Input:
All Java programmers program good programs.
Sample Output:
9 programmers
21 program
34 programs


Memory limit: 256 MB
Time limit: 8 seconds
