Thread-safe account

You have a class named Account. It has a long field storing the current balance. The account has three not-implemented methods:

  • deposit method adds an amount to the balance;
  • withdraw method subtracts an amount from the balance if the account has enough money. The method must return true if the operation is successful, otherwise, it returns false;
  • getBalance method returns the current balance.

Implement the given methods. The class should work correctly with multiple threads. The testing system will create threads and call methods on the same instance of Account.

Do not make the class public.
