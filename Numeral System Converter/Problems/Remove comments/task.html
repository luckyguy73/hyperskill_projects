Remove comments Open on JetBrains Academy

Given a line of Java code. Of course, this line was written by good programmers, so they left a lot of comments here. But now, someone wants to actually understand how this code works, but these comments happen to be pretty uninformative and describe obvious things. Also, they interfere with reading the code.

Your task is to delete all these ugly useless comments from code.

Programmers wrote "/* this type of comments */" as well as "// this type of comments".

All comments of these programmers are considered bad and useless so you need to delete all of them.


Sample Input:
Integer variable /* it only contains integers */= new Integer("0"); // now contains zero
Sample Output:
Integer variable = new Integer("0");

Sample Input:
/* new line */String /* we need String, not int */s /* bad name, I think */= "123";
Sample Output:
String s = "123";

Sample Input:
int b = 0;/* expression */ b = b /* **/* b /* b * b = b squared */; // 0 * 0 = 0
Sample Output:
int b = 0; b = b * b ;


Memory limit: 256 MB
Time limit: 8 seconds
